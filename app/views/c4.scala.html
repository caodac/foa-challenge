The blackboard architecture, an agent-based architecture, has been proposed as the underlying architecture for the eventual implementation of the Biomedical Translator during the feasibility phase. While far from complete, this codebase provides an initial proof-of-concept implementation. The goal of this challenge is to give the participant an opportunity to be familiarized with the blackboard codebase.

The task for this challenge is as follows:
  1. Clone the repository https://spotlite.nih.gov/ncats/blackboard.git
  2. Build and run the code based on the provided instructions
  3. Let's proceed to construct a knowledge graph with the query term mast cell as the seed. While there are multiple knowledge sources available, for the purpose of this exercise we're restricting the knowledge graph to be built with only the Pharos knowledge source.
